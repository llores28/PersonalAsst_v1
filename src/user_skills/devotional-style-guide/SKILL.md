# Devotional Style Guide

## User Preferences

The user prefers devotionals with these characteristics:
- **Length**: 5-10 minutes of reading time (approximately 500-800 words)
- **Tone**: Deep-analysis biblical studies style
- **References**: NIV Bible + Ellen G. White writings when relevant
- **Style**: Non-denominational, unbiased language, no personal opinion
- **Format**: Scripture passage, commentary, reflection question

## Structure Template

Each devotional should follow this structure:

### 1. Opening Scripture (1-2 verses from NIV)
Select a scripture that speaks to the day's theme. Include the reference.

### 2. Context (2-3 sentences)
Provide brief historical or cultural background to help understanding.

### 3. Commentary (3-4 paragraphs)
This is the main teaching section:
- Explain the scripture's meaning using only the bible
- Connect to everyday life
- Include practical application
- Use references to Ellen G. White insights to add depth to the topic

### 4. Reflection Question
One open-ended question for personal application. Example formats:
- "How does this passage challenge your current perspective on...?"
- "In what area of your life do you need to apply this truth?"
- "What would it look like to live out this principle today?"

### 5. Closing thoughts
2-3 sentences that:
- The revealed truth
- How to apply it in daily life
- End with hope/encouragement

## Writing Guidelines

### Do:
- Use inclusive language ("we", "us", "believers")
- Ground insights in scripture
- Make practical applications specific
- Vary sentence structure for readability
- Include a memorable takeaway

### Don't:
- Use denominational-specific language
- Engage in theological debates
- Use clichés ("God is good all the time")
- Reference current politics or news
- Preach or use judgmental tone
